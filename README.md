# Walmart-sales-data-analysis

# Purpose
The project walmart sales data analysis using MySQL aims to understand business insights and trends related to sales performance,customer activity and operational insights. This dataset contains 995+ records and they were used for data cleaning, feature engineering and explanatory data analysis. 

# Objectives
The main objectives of this data analysis are:- 
 1. Analyse overall sales performance
 2. Identify top-performing product lines 
 3. Compare sales across different branches
 4. Study customer purchasing behaviour
 5. Understand revenue trends
 6. Generate business insights using SQL

# Dataset information
 The dataset contains walmart sales transactional sales data with information such as:-

 8. quantity
 9. VAT
